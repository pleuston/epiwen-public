# Epigraphic Literature (金石學) — staged data

Ingested from the obsidian-vault skslxb harvest (`build_literature.py`).

- `works.json` — 36 works (22 parsed catalogues + 14 gazetteers/字書).
- `toc/<id>.json` — per-work `{excerpts, inscriptions}` (the 目錄).
- `authorities.json` — 1698 cross-catalog inscription concordance records (+ Wikidata for 39).

**Provenance / licensing:** texts from **Kanripo** (四庫全書 文淵閣 WYG + 四部叢刊 SBCK; open) and **石刻史料新編 / SKQS** (public-domain); cross-catalog authorities derived by the Epiwen project; **Wikidata** enrichment under CC-BY-SA (attribution required). 六藝之一錄 is an aggregating compendium (its 5,258 titles overlap the other catalogues).
